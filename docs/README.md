# WHU Circle 文档索引

本目录按“当前设计、接口约定、联调过程、实践报告、截图证据、历史归档”分类。业务实现以源码和当前设计文档为准；`archive/` 仅用于追溯项目过程，不作为当前接口依据。

## 当前设计

- [`design/系统设计说明书.md`](design/系统设计说明书.md)：当前系统架构、功能、算法、类图、数据库、UI、安全与技术栈说明。
- [`design/系统设计说明书.html`](design/系统设计说明书.html)：设计说明书的 HTML 版本。

## API 文档

- [`api/WHU_Circle_后端API开发清单.md`](api/WHU_Circle_后端API开发清单.md)：前后端接口和联调总览。
- [`api/活动与日历_API_v1.md`](api/活动与日历_API_v1.md)：频道活动、报名、日历和活动群聊设计。
- [`api/群聊管理_API_v1.md`](api/群聊管理_API_v1.md)：群聊管理权限与接口。

## 联调与过程记录
